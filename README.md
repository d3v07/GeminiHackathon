# Metropolis: Real-Time NPC Simulation Engine

Autonomous AI agents living in a city. Procedural routines. Memory. Cognition.

## Core
Temporal.io workflows + Gemini 2.0 Flash + LangGraph + Pinecone vector memory + Next.js UI.

## Quick Start
```bash
DEMO_MODE=true docker compose up --build
# Open http://localhost:3000
```
